supplier开发文档编写

首页：

1、查询所有供应商

入参：year String 年份

SELECT supplier,SUM(money) FROM ods_supplier_ys WHERE YEAR = ‘2020’ GROUP BY supplier
ORDER BY SUM(money) DESC

2、根据供应商名字查工厂

入参：year String 年份 ，s_name

SELECT ofy.factory,SUM(ofy.money) FROM supplier_factory sf ,ods_factory_ys ofy WHERE sf.s_name=’福州福森’ AND ofy.factory=sf.f_name
AND ofy.year=’2020’ GROUP BY ofy.factory ORDER BY SUM(ofy.money) DESC;

3、根据工厂名字查货号

SELECT goods_sku,SUM(money) FROM ods_factory_ys WHERE factory=’寿光佳泰’
AND YEAR=’2020’ GROUP BY goods_sku ORDER BY SUM(money) DESC;
